##第一章 解决问题与程序设计竞赛   
**1. FESTIVAL问题**    
有最大可租用天数N，以及最小需租用天数L，在需连续租用的前提下，已知连续N天的每日租金，为使平均日租金降至最低，应如何租用？  
问题见书本P5
   
**输入：**      
"用例个数C(C<=100)；
针对每个用例，有两行输入：  
Line 1：可租用最大天数N(1<=N<=1000)、最少需连续租用天数L(1<=L<=1000, L<=N);  
Line 2：N个日租金值(租金值为<100的自然数)"   

**输入示例：**            	
2   
6 3   
1 2 3 3 2 1   
6 2     
1 2 3 3 2 1   

**输出：**       	
对应各测试用例，每一行显示一个最小平均日租金值(允许小于10-7的绝对/相对误差

**输出示例：**      	
1.7500000000   
1.5000000000

**运行时间/内存限制：**      
Time < 2s   
Mem <= 64MB
